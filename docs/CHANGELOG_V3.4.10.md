# CHANGELOG v3.4.10

## Outlier Score 확장

상위 10% 보너스 판정 항목을 3개에서 5개로 확장했습니다.

- 라인 점수
- 챔피언 폭
- 티어 점수
- 내전 승률
- 내전 KDA

## 상위 퍼센타일 처리

- 컷오프 값과 동점인 멤버는 모두 상위 그룹에 포함합니다.
- Java의 `isTopPercentile(value, threshold)` 헬퍼로 명시했습니다.

## 승률과 KDA 기준

- 내전 승률: 전체 내전 승리 수 / 전체 내전 경기 수
- 내전 KDA: `(평균 킬 + 평균 어시스트) / max(1, 평균 데스)`
- 경기 기록이 없는 멤버: 승률 0.5, KDA 3.0
- 승률/KDA는 z-score 가중합에는 직접 넣지 않고 Outlier 보너스에만 사용합니다.
