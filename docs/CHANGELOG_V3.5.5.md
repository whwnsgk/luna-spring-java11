# CHANGELOG v3.5.5

## 자동 팀 배정 라인 표시

자동 팀 짜기는 기존부터 TOP/JUNGLE/MID/ADC/SUPPORT 라인 배치를 함께 전수 검사했습니다.
이번 버전에서는 그 결과를 팀 카드 앞에 명확하게 표시합니다.

- 탑
- 정글
- 미드
- 원딜
- 서폿

경매 화면은 기존 방식 그대로 유지합니다.

## 개인별 산식 적용 내역

자동 팀 결과 아래에 별도 상세 영역을 추가했습니다.

- 배정된 라인
- 해당 라인 숙련도
- 기본 레이팅
- 포지션 과밀 페널티
- 최종 유효 레이팅
- 1점/3점/0점 배치 페널티 여부
- 각 맞라인 레이팅 차이
- 전체 알고리즘 평가점수

## 포지션 과밀 기준 수정

현재 라인 숙련도 정의에 맞게 과밀의 주라인 기준을 2점이 아니라 3점으로 수정했습니다.

- 3점: 다이아 이상 주라인
- 2점: 솔랭 가능
- 1점: 자랭/5인큐 가능
