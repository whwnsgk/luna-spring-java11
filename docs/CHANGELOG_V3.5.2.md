- 0~3 라인 숙련도
- 2점 우선, 1점 보조, 3점 비상 해제, 0점 금지
- 맞라인 Elo 차이 중심 평가
- 양 팀 총점 차이와 각종 페널티
- 최고 tier_score 멤버 SUPPORT 제한
- 상위 5개 추천
- 자동 배정 실패 수정 안내
- 강제 팀 짜기
- 경기 상세 밸런스 투표
- 투표 기반 가중치 추천과 적용
