# CHANGELOG v3.5.9

## 포지션 과밀 기준 변경

과밀 집계 대상을 3점 주라인만이 아니라 다음 두 등급으로 확대했습니다.

- 3점: 다이아 이상 주라인
- 2점: 솔랭 가능

한 사람이 여러 2점/3점 라인을 설정하면 해당 라인을 모두 집계합니다.
라인별 가능 인원이 2명을 초과하면 초과 인원당 50점 기준 페널티를 계산합니다.

- 3점 라인: 기준 페널티 100%
- 2점 라인: 기준 페널티 70%
- 다른 대체 라인이 있으면 추가 완화
- 여러 과밀 라인이 있으면 가장 큰 페널티 하나만 적용

## 컷씬 음성 안정화

기존 문제:

- 라인 전환 시 이전 Audio의 play()가 완료되기 전에 pause()가 호출되어 AbortError 발생
- 파일 후보를 매번 순차 로드하면서 재생 성공률이 낮았음

변경 후:

- 페이지 로딩 시 음성 파일을 미리 fetch하여 Blob URL로 캐시
- 각 라인마다 새로운 Audio 객체로 1회 재생
- 라인 전환 시 현재 음성만 정상 종료
- AbortError는 정상적인 중단으로 처리
