# CHANGELOG v3.2

## 역할 체계 변경

실시간 경매 역할을 네 가지로 분리했습니다.

- 진행자(DIRECTOR)
- 블루 호스트(BLUE)
- 레드 호스트(RED)
- 관전자(SPECTATOR)

방을 만든 사람은 자동으로 진행자로 접속합니다. 진행자는 호스트 자리를 차지하지 않습니다.

## 경매 시작 권한

- 경매 시작 버튼은 진행자에게만 표시됩니다.
- 진행자, 블루 호스트, 레드 호스트가 모두 접속한 상태에서만 활성화됩니다.
- 호스트와 관전자는 경매를 시작할 수 없습니다.

## 대기 → 시작 연출 → 진행

경매 상태가 다음 순서로 전환됩니다.

1. WAITING: 참가자 접속 대기
2. STARTING: 약 4초 동안 전체 화면 시작 애니메이션
3. RUNNING: 첫 매물 공개 및 60초 경매 시작
4. COMPLETE: 경매 완료

시작 애니메이션은 모든 접속자 화면에 동시에 표시됩니다.

## 접속 현황

상단 상태표시에 다음이 실시간 표시됩니다.

- 진행자 접속 여부
- 블루 호스트 이름과 접속 여부
- 레드 호스트 이름과 접속 여부
- 관전자 수
