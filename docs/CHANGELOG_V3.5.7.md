# CHANGELOG v3.5.7

## 포지션 과밀 표시 개선

기본 점수와 유효 점수가 같은 경우에도 이유를 명확하게 표시합니다.

- `3점 주라인 미설정 → 과밀 보정 대상 아님`
- `미드 3점 주라인 2명 · 기준 2명 이하 → 과밀 보정 없음`
- `서폿 3점 주라인 4명 · 기준 2명 초과 → 과밀 -100`

현재 과밀 기준은 라인 숙련도 3점인 주라인 인원입니다.

## 라인 음성 파일 호환성

라인별 음성을 여러 확장자와 위치에서 순차 탐색합니다.

- MP3
- WAV
- M4A
- OGG
- `/media/`
- 정적 리소스 루트 `/`

모든 후보가 404일 경우 콘솔에 확인한 경로를 경고로 출력합니다.
