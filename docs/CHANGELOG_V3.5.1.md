# CHANGELOG v3.5.1

## 시네마틱 사운드

- 시네마틱 영상을 음소거 없이 재생하도록 변경
- 브라우저가 소리 재생을 차단할 경우 음소거 재생으로 자동 대체
- 자동 팀 짜기 버튼이나 경매 입찰처럼 사용자 동작 뒤에는 소리 포함 재생 가능

## 경매 낙찰 연출 순서

1. 기존 `누구 낙찰!` 안내 표시
2. 안내가 사라진 뒤 약 2.45초 후
3. `완벽한 입찰!` 또는 `너무 비싼 입찰!` 영상 재생

두 연출이 겹치지 않습니다.

## 자동 팀 짜기 실패 안내

유효한 팀을 만들 수 없을 때 다음 내용을 표시합니다.

- 라인별 가능한 인원 수 분석
- 가장 빠르게 수정할 라인 안내
- 라인 설정 화면으로 돌아가기
- 강제로 팀 짜기

## 강제 팀 짜기

강제 진행 시 마지막 수단으로 다음 제한을 해제합니다.

- 0점 라인 배치 금지
- 최고 티어 멤버의 서포터 배치 금지

강제 후보에서도 0점 배치와 최고 티어 서포터는 매우 큰 페널티를 받아 가능한 한 적게 사용됩니다.
