# CHANGELOG v3.0.1

## 토큰 보정 규칙 수정

기존:
- 강한 호스트 기본 토큰 유지
- 약한 호스트에게 보정 토큰 추가

수정:
- 약한 호스트는 기본 토큰 유지
- 강한 호스트의 시작 토큰에서 `코스트 차이 × 보정 비율` 차감

예시:
- 블루 호스트 1,605
- 레드 호스트 1,461
- 기본 토큰 1,000
- 보정 비율 0.5
- 차감량 72
- 블루 928 / 레드 1,000

## 내전 짜기 → 경매 이동 개선

- 참가자 영역에 `10명으로 바로 경매` 버튼 추가
- 참가자 10명이 모이면 즉시 활성화
- 참가자가 선택 풀 또는 블루/레드 팀 어디에 있든 합계 10명으로 판정
- 숫자/문자열 멤버 ID 혼용으로 버튼이 비활성화되던 문제 수정
- 기존 중앙 버튼도 동일한 조건으로 정상 활성화

## 경매 설정 UI 개선

- 라벨과 입력 요소를 카드형 3열 레이아웃으로 정리
- 입력 폭을 통일
- 호스트/기본 토큰/보정 비율 변경 시 예상 시작 토큰 즉시 표시
- 모바일 및 좁은 화면 반응형 보완
