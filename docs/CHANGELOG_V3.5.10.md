# CHANGELOG v3.5.10

## 예상 승률 완화

기존에는 양 팀 5명의 총점 차이를 개인 Elo 공식에 그대로 넣어
8% 대 92%처럼 지나치게 극단적인 승률이 표시될 수 있었습니다.

변경 후:

```
블루 평균 레이팅 = 블루팀 총점 / 5
레드 평균 레이팅 = 레드팀 총점 / 5

블루 예상 승률
= 1 / (1 + 10 ^ ((레드 평균 - 블루 평균) / 400))
```

자동 팀 짜기와 경매 종료 예상 승률에 동일하게 적용됩니다.
