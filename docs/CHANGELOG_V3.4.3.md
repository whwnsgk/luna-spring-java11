# CHANGELOG v3.4.3

## 경매 입찰 흐름 변경

- 어필 입력/전송 및 어필 표시 제거
- BLUE/RED 호스트에 직접 입찰 금액 입력칸 추가
- 새 매물 공개 후 첫 30초는 양 팀 누구나 첫 금액 입력 가능
- 먼저 입찰한 팀의 상대 팀부터 다음 차례 진행
- 첫 30초 무입찰이면 유찰
- 첫 30초에 양 팀 모두 포기하면 유찰
- 유찰된 매물은 남은 후보 대기열 맨 뒤로 이동
- 첫 입찰 후에는 기존 교대 입찰 방식 유지
- 교대 입찰 중 현재 차례 팀이 포기하면 최고 입찰 팀에 즉시 낙찰

## 매물 정보

- 매물 카드에 해당 멤버가 내전에서 가장 많이 사용한 챔피언 표시
- 챔피언 이미지는 Riot Data Dragon 이미지 사용
