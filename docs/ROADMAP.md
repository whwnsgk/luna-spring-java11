# Rift Arena 로드맵

## 1차 완료
- 멤버 CRUD, 10명 드래그 앤 드롭, 자동/수동 편성
- 경기 결과 수동 등록, 승률/KDA/MVP 통계

## 2차 현재 구현
- Riot ID → PUUID → 소환사/솔랭 정보 자동 갱신
- 솔랭 최근 20경기 승패, 승률, 평균 K/D/A 계산
- 최근 모스트 라인 및 챔피언 3개
- Riot 스냅샷 이력 저장
- 멤버 상세 모달
- 내전 모스트 챔피언
- 같은 팀 궁합 최고/최악 계산
- Discord용 팀 구성 클립보드 복사

## 3차 예정
- 토큰 경매, 호스트 실력차 토큰 보정, 입찰 애니메이션
- 실시간 경매 상태 저장, 커스텀 경기 자동 탐색, 고급 승률 예측
