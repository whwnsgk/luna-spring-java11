# CHANGELOG v3.5.8

## 다중 3점 주라인 과밀 계산 수정

기존 문제:

- 한 사람이 3점 주라인을 여러 개 설정해도 첫 번째 라인 하나만 과밀 집계에 포함
- 실제로는 미드 3점 인원이 4명인데 탑/미드로 분산 집계되어 페널티가 누락될 수 있었음

변경 후:

- 한 사람의 모든 3점 주라인을 각각 라인별 인원수에 포함
- 해당 멤버가 가진 3점 주라인들의 과밀 페널티를 모두 계산
- 중복 합산하지 않고 가장 큰 페널티 하나만 적용

예시:

```
정호: TOP 3점, MID 3점
TOP 3점 인원 4명 → 기본 페널티 100
MID 3점 인원 3명 → 기본 페널티 50
최종 적용 → 100점 하나만 적용
```

다른 라인 가능 여부에 따른 완화:

- 다른 2점 라인 보유: 과밀 페널티 50% 적용
- 다른 1점 라인 보유: 과밀 페널티 70% 적용
- 대체 가능 라인 없음: 100% 적용

## 화면 표시

- 전체 TOP/JUNGLE/MID/ADC/SUPPORT 3점 인원수 표시
- 개인별 모든 3점 주라인 표시
- 어떤 라인의 과밀이 가장 커서 페널티가 적용되었는지 표시
