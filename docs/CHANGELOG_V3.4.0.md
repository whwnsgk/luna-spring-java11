# CHANGELOG v3.4.0

## 종합 개인 레이팅

개인 밸런스 점수는 다음 요소를 합산합니다.

- 솔로랭크 티어/단계/LP
- 최근 20경기 승률
- 최근 20경기 KDA
- 최근 20경기 모스트 챔피언 게임 수와 승률
- 내전 승률
- 내전 출전 수
- MVP 횟수

Riot 갱신 및 경기 저장/수정 후 자동으로 다시 계산합니다.

## 내전 짜기 BGM

- 내전 짜기 화면에 BGM 버튼, 볼륨, 상태 표시 추가
- 자동재생 차단 시 화면 클릭마다 재생 재시도
- MP3 파일 누락 여부 표시
- 다른 페이지로 이동하면 정지

## ClientAbortException

브라우저 새로고침, 페이지 이동 또는 오디오 요청 취소로 발생하는
ClientAbortException을 서버 장애 로그처럼 출력하지 않도록 처리했습니다.
