# CHANGELOG v3.5.3

## 강제 팀 짜기 서버 오류 수정

강제 팀 짜기에서는 최대 약 181만 개의 팀·라인 후보가 생성될 수 있습니다.

이전 구현은 후보를 전부 메모리에 저장하여 무료 서버에서 메모리 부족 또는 서버 오류가 발생할 수 있었습니다.

변경 후:

- 모든 후보를 평가하되
- 가장 좋은 후보 100개만 우선순위 큐에 유지
- 최종적으로 중복을 제거하고 상위 5개 추천 출력

따라서 강제 팀 짜기에서도 메모리 사용량이 크게 줄어듭니다.
