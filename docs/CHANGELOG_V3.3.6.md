# CHANGELOG v3.3.6

## 시즌 생성 안정화

- 시작일 필수 검증
- 종료일이 시작일보다 빠른 경우 차단
- 날짜 문자열을 Java `LocalDate`로 변환 후 JDBC DATE로 저장
- 활성 시즌이 하나도 없으면 새 시즌을 자동 활성화

## 경기 기록 저장 안정화

- seasonId가 없으면 활성 시즌을 재조회
- 활성 시즌도 없으면 DB 오류 대신 안내 메시지 반환
- 시즌이 없을 때 일반/강제 경기 기록창을 열지 않고 시즌 생성창 표시

## 챔피언 자동완성

- 강제 경기 기록과 일반 경기 기록의 챔피언 입력란에 자동완성 추가
- 한글/영문 모두 검색 가능
- 한글 챔피언명을 선택해도 저장 시 Riot 영문명으로 정규화
