# CHANGELOG v3.5.0

## 라인 숙련도

- 3: 다이아 이상 수준의 강한 주라인
- 2: 솔랭 플레이 가능
- 1: 자랭·5인큐 플레이 가능
- 0: 절대 배치 금지
- 각 점수는 여러 라인에 중복 설정 가능

## A안 배치 정책

1. 2점과 1점만으로 먼저 유효한 팀을 탐색
2. 2점 배치를 우선하고 1점은 보조 사용
3. 유효한 팀이 하나도 없을 때만 3점 주라인을 비상 해제
4. 0점 라인은 어떤 경우에도 배치하지 않음
5. 선택된 10명 중 tier_score 최고 멤버는 SUPPORT 배치 금지

## 자동 팀 추천

평가값이 낮을수록 좋은 조합입니다.

- 맞라인 Elo 차이 합
- 양 팀 총점 차이
- 1점 배치 페널티
- 3점 비상 배치 페널티
- 포지션 과밀 페널티

상위 5개 추천을 화면에 표시하고 사용자가 선택할 수 있습니다.

## 경기 밸런스 투표

경기 상세 화면 하단에서 황밸/보통/나빠요 투표가 가능합니다.
브라우저별 voter_key를 사용하여 같은 경기의 투표를 수정할 수 있습니다.

## 가중치 튜닝

경기 생성 당시의 밸런스 특성을 저장합니다.
투표가 5경기 이상 쌓이면 황밸 경기와 나빠요 경기의 특성을 비교해 추천 가중치를 계산합니다.
통계 화면에서 추천값을 확인하고 `추천 가중치 적용` 버튼으로 반영합니다.
