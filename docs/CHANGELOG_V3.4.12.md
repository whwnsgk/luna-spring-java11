# CHANGELOG v3.4.12

## 라인 입력 구조 변경

기존의 선호 포지션 체크박스를 제거하고, 사람별·라인별로 다음 값을 입력합니다.

- 선호도 2: 1순위
- 선호도 1: 2순위
- 선호도 0: 절대 안 감
- 기용 가능한 챔피언 수: 0 이상의 직접 입력값

한 사람은 1순위가 정확히 하나여야 하며, 2순위는 최대 하나입니다.

## 라인 점수

라인별 채용 가능한 인원 수를 `n_lane`으로 계산합니다.

```
개인 라인 점수 = Σ (500 / n_lane) × lane_indicator
```

## 챔피언 폭

경기 기록의 고유 챔피언 수를 자동 집계하지 않고 수동 입력값을 사용합니다.

```
챔피언 수 <= 7 : 입력값 그대로
챔피언 수 > 7  : 7 + log(챔피언 수 - 6)
```

## 수동 티어와 Riot 갱신

- 멤버별 `수동 티어 고정` 여부 저장
- 일괄 갱신 전에 대상 목록과 체크박스 표시
- 수동 티어 멤버는 `티어 덮어쓰기`를 체크해야 Riot 실제 티어로 초기화
- 체크하지 않으면 수동 티어/랭크/LP는 보존하고 나머지 Riot 통계만 갱신

## 점수 재계산

수동 티어, 라인 선호도, 챔피언 수를 저장하면 전체 레이팅을 즉시 재계산합니다.

## 외부인

- Riot ID 없이 외부인 등록 가능
- 수동 티어, 라인 선호도, 챔피언 수로 초기점수 계산
- Riot API 갱신 대상에서는 제외
