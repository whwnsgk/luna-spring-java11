# CHANGELOG v3.4.11

## 초기 Elo 산식 변경

기존의 전체 z-score 기반 1500/200 정규화를 제거했습니다.

새 최초 Elo:

```
tier_score + lane_score + champ_score
```

통계 데이터 반영:

```
최초 Elo
+ z(내전 통산 승률) × 100
+ z(내전 통산 KDA) × 60
+ outlier_score × 20
```

- 승률과 KDA의 100:60 비율은 기존 요구 가중치 0.25:0.15 비율을 점수로 환산한 값입니다.
- 경기 기록이 없는 멤버는 승률 0.5, KDA 3.0을 사용합니다.
- Outlier는 티어/라인/챔피언 폭/승률/KDA 상위 10% 항목 개수입니다.

## 경기 Elo 갱신 변경

각 선수의 기대 승률은 다음 두 값을 50%씩 반영합니다.

1. 동일 포지션 맞라이너와의 Elo 기대 승률
2. 양 팀 평균 Elo 기준 기대 승률

```
최종 기대 승률 = (맞라인 기대 승률 + 팀 기대 승률) / 2
변동값 = 32 × (실제 승패 - 최종 기대 승률)
```

따라서 자신과 팀이 상대보다 강한 조건에서는 승리 시 소폭 상승하고 패배 시 크게 하락합니다.
약한 조건에서는 승리 시 크게 상승하고 패배 시 적게 하락합니다.
