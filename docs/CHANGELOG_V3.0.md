# CHANGELOG v3.0 — Token Auction + Web Audio

## 토큰 경매 MVP

- 내전 짜기에서 선택한 10명을 경매장으로 전달
- 호스트 두 명 직접 선택
- 높은 코스트 호스트 자동 추천
- 약한 호스트 토큰 자동 보정
- 낮은 밸런스 코스트 후보부터 경매
- 블루/레드 입찰 버튼
- 카운트다운 후 자동 낙찰
- 유찰 시 후보 후순위 이동
- 한 팀이 5명에 도달하면 남은 선수 자동 배정
- 경매 완료 시 내전 짜기 화면에 최종 팀 자동 반영
- 경기 기록 및 Discord 복사로 연결
- localStorage를 이용한 진행 상태 복원

## Web Audio API

별도 MP3/WAV 파일 없이 브라우저의 Web Audio API로 아래 효과음을 생성합니다.

- 경매 시작
- 후보 등장
- 블루/레드 입찰
- 카운트다운
- 블루/레드 낙찰
- 유찰
- 오류
- 팀 완성 팡파르

음소거와 볼륨 설정은 브라우저 localStorage에 저장됩니다.
