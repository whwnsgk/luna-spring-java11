# CHANGELOG v3.3

## 턴제 입찰

- 매물마다 첫 입찰 호스트를 BLUE/RED 중 무작위로 선정
- 선정된 호스트가 기본금액 10원을 자동 입찰
- 이후 상대 호스트부터 번갈아 입찰
- 자신의 턴에만 입찰 및 포기 가능
- 입찰 후 상대 호스트에게 턴 자동 전달
- 각 턴 제한시간 60초
- 시간 초과 또는 포기 시 현재 최고 입찰자가 즉시 낙찰

## 애니메이션

- 첫 선공자 이름과 10원 자동 입찰 표시
- 매 턴마다 실제 호스트 이름으로 TURN 안내
- 낙찰 시 구매한 호스트 이름, 매물 이름, 낙찰 금액 표시
- 마지막 경매 종료 시 전체 화면 AUCTION COMPLETE 연출
