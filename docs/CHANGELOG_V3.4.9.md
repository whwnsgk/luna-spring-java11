# CHANGELOG v3.4.9

## 포지션 과밀 개인 점수 보정

- 선택된 10명의 1순위 포지션을 집계
- 포지션당 정상 인원은 2명
- 초과 인원 1명당 기본 50점 감점
- 다른 선호 포지션 보유 시 70% 적용
- FILL 보유 시 30% 적용
- 자동 팀 짜기와 토큰 경매에서 동일한 유효 레이팅 사용
- DB의 `member.balance_score`는 변경하지 않음
- 자동 팀 짜기 화면에 `과밀 -점수` 및 조정된 코스트 표시
- 경매방 로그와 매물 코스트에 조정 결과 반영

## 예시

SUPPORT 1순위가 4명이면 초과 2명으로 기본 페널티는 100점입니다.
- SUPPORT만 가능: -100
- SUPPORT + 다른 라인: -70
- SUPPORT + FILL: -30
