레이팅 계산 근거

1. 초기 파워 점수

  • 티어 점수 30% : 현재 솔로 랭크 티어/티어 단계(I, II, III, IV)를 점수화합니다.
  • 라인 적합도 15% : 선호 포지션 1순위는 2점, 2·3순위와 FILL은 1점으로 보고 5개 라인에 분배합니다.
  • 챔피언 폭 15% : 경기 기록 기준으로 포지션별 고유 챔피언 수를 계산합니다.
  • 상위 10% 보너스 : 티어 점수, 라인 점수, 챔피언 폭, 내전 승률, 내전 KDA가 각각 상위 10%에 들면 항목당 보너스 1점을 줍니다. 컷오프 동점자는 함께 포함됩니다.
  • 내전 승률/KDA 는 가중 z-score에 직접 더하지 않고, 상위 10% Outlier 보너스에만 반영합니다. 경기 기록이 없는 멤버는 승률 0.5, KDA 3.0을 사용합니다.

최초 Elo = tier_score + lane_score + champ_score

통계 반영 Elo = 최초 Elo + z(통산 승률)×100 + z(통산 KDA)×60 + outlier_score×20

2. 경기 결과 반영 방식

  • 저장된 경기들을 played_at → match_id 순서로 다시 읽습니다.
  • 같은 포지션끼리 TOP / JUNGLE / MID / ADC / SUPPORT 매치업을 잡습니다.
  • 맞라이너 Elo 기대 승률과 양 팀 평균 Elo 기대 승률을 각각 계산합니다.
  • 두 기대 승률을 50%씩 섞은 뒤 최종 팀 승패를 반영합니다.

맞라인 기대 승률 = 1 / (1 + 10 ^ ((상대 라인 점수 - 내 점수) / 400))

팀 기대 승률 = 1 / (1 + 10 ^ ((상대 팀 평균 - 우리 팀 평균) / 400))

최종 기대 승률 = (맞라인 기대 승률 + 팀 기대 승률) / 2

반영 점수 = 현재 점수 + 32 × (실제 결과 - 최종 기대 승률)

내 팀과 내 맞라이너가 상대보다 강하면 승리 시 소폭 상승하고 패배 시 크게 하락합니다. 반대로 약한 조건에서 이기면 크게 상승하고 패배 시 적게 하락합니다.

3. 선택 인원 포지션 과밀 보정

  • 선택된 10명에서 각 1순위 포지션의 정상 인원은 2명으로 봅니다.
  • 3명 이상이면 (초과 인원 × 50점)을 해당 포지션 1순위 멤버의 그 내전용 점수에서 차감합니다.
  • 다른 선호 포지션이 있으면 페널티의 70%, FILL이 있으면 30%만 적용합니다.
  • 이 유효 점수는 자동 팀 짜기와 토큰 경매에 모두 적용되며 DB 원본 레이팅은 바뀌지 않습니다.

유효 점수 = 기본 레이팅 - (초과 인원 × 50 × 포지션 유연성 배율)

4. 팀 자동 배정 기준

  • 10명 중 5명을 고르는 팀 조합 126개를 전수 검사합니다.
  • 각 팀의 5개 라인 배치 120개를 다시 전수 검사합니다.
  • 라인 적합도 합 × 50 - 양 팀 총 레이팅 차이 값이 가장 좋은 조합을 채택합니다.

그래서 단순 평균만 맞추는 것이 아니라, 각 팀이 실제로 라인을 소화할 수 있는지까지 함께 고려합니다.

5. 기존 데이터 반영

  • 기존 경기 데이터를 지울 필요는 없습니다.
  • 멤버 페이지의 레이팅 전체 재계산 버튼을 누르면, 현재 저장된 모든 경기 결과를 기준으로 코스트를 다시 계산합니다.
  • 새 경기 등록 또는 기존 경기 수정 후에도 전체 레이팅이 다시 계산됩니다.
